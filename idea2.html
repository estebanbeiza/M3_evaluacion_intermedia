A quien le preste esto?

Te has preguntado a quien le has prestado algo y no lo recuerdas? si la respuesta es sí, entonces ya somos dos.
Ahora piensas, no hubiese sido mejor haber registrado a quien le prestaste algo y de esa forma, aunque tu memoria lo olvidase ya no importaría, porque lo tendrías registrado.

Para ello nacerá nuestra aplicación "A Quién le preste esto?". basandanos en un modelo simple al inicio lo desarrollaremos utilizando una base de datos SQL para almacenar los registros con java crearemos la interfaz para realizar la gestion de los datos (CRUD) y con HTML5, CSS y JavaScript una interfaz para la interaccion del usuario con la aplicacion.

Tal como su nombre lo indica ayudará a recordar a toda persona a quien le presto alg ya que al momento del prestamo lo registrará en nuestra aplicación de esta forma aqune pase mucho tiempo y nuestra memoria olvide algunos hechos la aplicación ayudara a recuperar ese prestamo que al día de hoy se da por perdido por el hecho de no recordar a quien se lo presté. principalmente
