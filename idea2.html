A quien le preste esto?

Problematica

Te has preguntado a quien le has prestado algo y no lo recuerdas? si la respuesta es sí, entonces ya somos dos.
Ahora piensa, no hubiese sido mejor haber registrado a quien le prestaste algo y de esa forma, aunque tu memoria lo olvidase, ya no importaría, porque lo tendrías registrado.

Para ello nacerá nuestra aplicación A Quién le preste esto?. basandanos en un modelo simple al inicio lo desarrollaremos utilizando una base de datos SQL para almacenar los registros con java crearemos la interfaz para realizar la gestion de los datos (CRUD) y con HTML5, CSS y JavaScript una interfaz para la interaccion del usuario con la aplicacion.

Tal como su nombre lo indica ayudará a recordar a quien le presto algo ya que al momento del prestamo lo registrarás en nuestra aplicación de esta forma aunque pase mucho tiempo y nuestra memoria olvide algunos hechos la aplicación ayudara a recuperar ese prestamo que hoy se das por perdido.
